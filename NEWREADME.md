## What's in this mod? :3

### Cat Ears, Tails, Paws & Toe Beans >w<

A full set of dyeable cat accessories — now with **four wearable pieces** instead of two!

| Item | Slot | Accessories Slot |
|------|------|-----------------|
| Cat Ears | Helmet | Hat |
| Tail | Leggings | Belt |
| Paws | Chestplate | Gloves |
| Toe Beans | Boots | Feet |

All four pieces support **two-color dyeing** — a main color and an accent color — using dyes in a crafting table. Mix and match however you like :3

> tip: you can get fur by right-clicking a cat with a brush :3

**Crafting path** (all processed through Create):
- Headband → Incomplete Cat Ears → Cat Ears
- High Quality Fabric → Incomplete Tail → Tail
- Mittens → Incomplete Paws → Paws
- Socks → Incomplete Toe Beans → Toe Beans

Wear the **full set** (ears + tail + paws + toe beans) for extra fun :3

# Getting Started

So you've installed Create: MelatOwOin and a working Create setup. Here's the natural progression from "no fur" to "wearing a full dyed cat outfit".

## 1. Get Fur

Right-click a Cat with a vanilla **Brush**. You'll receive 1 Fur per interaction. The brush takes durability damage.

> The `docat` game rule controls this — see [[Tags-and-Game-Rules]].

The interaction also works through a **Create Deployer**, so an automated cat farm is possible.

## 2. Make Plastic

Plastic is the second universal ingredient. In a **Mixer**:

- 1× Dried Kelp
- 1× Andesite Alloy

→ 1× Plastic.

## 3. Make High Quality Fabric

Set up a **Sequenced Assembly** on white carpet (10 loops). Each loop runs:

1. Deploy Fur onto the carpet
2. Deploy Plastic onto the result
3. Press the result

Output: 1× High Quality Fabric.

## 4. Craft the Intermediate Wearables

In a vanilla crafting table:

- **Socks** — 2×2 of High Quality Fabric
- **Mittens** — two High Quality Fabric in a row
- **Headband** — iron-nugget border around one High Quality Fabric (see [[Recipes-Crafting-Table]])

## 5. Assemble the Cat Pieces

Set up **four Sequenced Assemblies** (or reuse one with switchable inputs). Each runs 3 loops of *Deploy Fur → Deploy Plastic → Press*:

| Base item | Output |
|-----------|--------|
| Headband | Cat Ears |
| High Quality Fabric | Tail |
| Mittens | Paws |
| Socks | Toe Beans |

## 6. Wear and Dye

Equip them like armor (Cat Ears = helmet, Paws = chestplate, Tail = leggings, Toe Beans = boots). See [[Wearable-Cat-Pieces]].

Dye them in a crafting table — see [[Dyeing-System]].

## 7. (Optional) The Chemistry Track

If you want chloroform, the Eepy effect, or the Orange Pill, work through [[Chloroform-Chain]] and [[Furrycore-Chain]] next.

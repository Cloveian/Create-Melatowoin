# Items

Every item the mod registers, grouped by purpose.

## Crafting Materials

| ID | Name | Max Stack | Notes |
|----|------|-----------|-------|
| `melatowoin:fur` | Fur | 16 | Drops from brushing cats. Repair material for cat equipment. |
| `melatowoin:strings` | String Mess | 64 | Tooltip: *weird ball of strings*. From milling white carpet. |
| `melatowoin:dust` | Dust | 1 | Tooltip: *dusty*. From milling String Mess. |
| `melatowoin:plastic` | Plastic | 64 | Mixed from dried kelp + andesite alloy. |
| `melatowoin:fabrics` | Incomplete Fabric | 1 | Transitional item for the High Quality Fabric sequence. |
| `melatowoin:fabric` | High Quality Fabric | 1 | Tooltip: *better than wool*. Base for cat-piece crafting. |
| `melatowoin:ca` | Calcium Acetate | 16 | Tooltip: *We need to cook*. Mixer output from vinegar + egg. |
| `melatowoin:purefurryness` | Pure Furrynes | 64 | Tooltip: *"95% pure"*. Used in Orange Pill recipe. |
| `melatowoin:unpurefurryness` | Unpure Furrynes | 64 | Tooltip: *you will regret going this path*. |
